# Sales-Performance-Dashboard_PowerBI
A Power BI dashboard built from multiple connected datasets to analyze company sales, product trends, customer distribution, and performance across years.

Highlights
- Tracks total sales, profit margin, top products, and product lines.
- Shows year-wise and product-wise sales and quantities.
- Includes top 5 and bottom 5 selling products.
- Displays customer locations on an interactive map.
- Offers a detailed-view page for deeper breakdowns.
- Includes a customer dashboard and a minimal version (CustDashMin).
- Features a tooltip page for quick product insights.
- Uses proper relationships and transformed tables for accurate reporting.
